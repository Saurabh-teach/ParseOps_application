# ParseOps Software Requirements Specification

**Project Name:** ParseOps  
**Version:** 1.0, prepared from current repository state  
**Prepared From:** Existing Django and React repository  
**Technology Stack:** Django 5, Django REST Framework, SimpleJWT, Channels, Celery, Redis, PostgreSQL, React, Vite, Axios, FullCalendar, Recharts

This documentation set is a reverse-engineered Software Requirements Specification (SRS) for the current ParseOps application. It documents observed behavior, inferred business rules, current implementation gaps, and recommended improvements. It does not prescribe source-code changes.
